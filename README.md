# Konpanion
 
### Konpanion is a forked HKMPpouch mod that adds a little knight to keep you company (Needs satchel, works with CustomKnight).

## How to add skins
**1. Download the latest release.**

**2. Go to the "skins" folder inside the Konpanion mod.**

**3. Create a new folder with any name you want.**

**4. Add any "Knight.png" file from any skin mod inside the skin folder.**

**Now your skin should show up under option "Custom"**
